# Cobalt Horizon Theme

Dive into deep coding sessions with **Cobalt Horizon**, a dark theme crafted on color science principles. It brings a balanced, high-contrast, visually engaging yet eye-friendly aesthetic to your editor.

## 🎨 Design Philosophy
Cobalt Horizon isn't just dark—it uses a rich deep-blue backdrop (`#19222C`) with carefully selected, luminous accent colors to make important syntax pop, without inducing eye strain. 

- **Background:** Rich, subdued deep blue/grey.
- **Keywords:** Subdue energetic Orange.
- **Functions:** Relaxed Golden Yellow.
- **Variables:** Vivid Cyan for high distinction.
- **Strings:** Bright, clear Green.
- **Constants:** Soft Pink for effortless scanning.

## ✨ Features
*   **High Contrast & Readability:** Optimized for long coding sessions and reducing visual fatigue.
*   **Minimalist UI:** The editor Chrome (Side Bar, Activity Bar, etc.) stays subtly out of your way while the active elements remain distinct.
*   **Modern Bracket & Indent Support:** Fully supports modern VS Code intent guide configurations out of the box.

## 🚀 Installation

### Option 1: VS Code Marketplace (Recommended)
1. Open the **Extensions** view in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`).
2. Search for `Cobalt Horizon`.
3. Click **Install**.
4. Click **Reload** to restart VS Code.
5. Go to **Preferences > Color Theme** (`Ctrl+K Ctrl+T` or `Cmd+K Cmd+T`) and select **Cobalt Horizon**.

### Option 2: Build From Source
1. Clone this repository: `git clone https://github.com/yourusername/cobalt-horizon.git`
2. Open the cloned folder in VS Code.
3. Press `F5` to open a new window with the theme loaded.
